拥有自己的博客🤡

　　现在生活节奏快，信息的获取范围广泛，作为一个刚入门的程序猿在对一些不懂得问题进行解决的时候往往会出现如下的样子（从开始搭建博客的相关博文最后调到了一个其他不想干的东西最后还是没有搞懂）如此这样的情况比比皆是，为了总结和反思。建立起自己的学习路径个方式方法不失为一个好的idea。所以搭建一个自己的博客就显得很重要了，当然博客并不是只是用来记笔记的，往往也是一种经历。

搭建前的准备工作👾

①环境的准备

  1. Node.js 官网
  2. Git 官网

②安装（官网的指南值得一看）

  1. 配置Github
    > 1. 注册请戳Github(已有请忽略此步)
    > 2. 创建代码库
    > 3. New repository
    > 4. 在Repository name下填写yourname.github.io(比如我这里就是ycomer.github.io)
    > 5. Setting
    > 6. 找到GIthub Pages
    > 7. 点击Automatic pagr generator
    > 8. 等待一会就欧克了。
  2. 配置hexo

    Item2官网

    打开item2 输入
    npm install hexo-cli -g


    上述完成以后输入
    npm install hexo –save


    查看是否安装完成输入
    hexo -v
  3. 初始化hexo

    输入
    hexo init folder
    　hexo会在folder(指定文件夹新建所需文件)

😄至此hexo搭建完毕
